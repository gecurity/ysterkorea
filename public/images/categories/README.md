# 카테고리 이미지 폴더

이 폴더에 각 제품 카테고리의 대표 이미지를 저장합니다.

## 필요한 이미지

다음 파일명으로 이미지를 추가해주세요:

1. **outdoor.jpg** - 실외용 LED 대표 이미지
2. **indoor.jpg** - 실내용 LED 대표 이미지
3. **special.jpg** - 특수 LED 대표 이미지
4. **rental.jpg** - 렌탈 LED 대표 이미지
5. **signage.jpg** - 간판 LED 대표 이미지

## 이미지 권장 사양

- **크기**: 800x600px 이상
- **비율**: 4:3 또는 16:9
- **형식**: JPG, PNG, WebP
- **용량**: 200KB 이하 권장 (웹 최적화)
- **내용**: 각 카테고리를 대표하는 LED 디스플레이 실제 사진

## 예시

```
outdoor.jpg  → 실외 빌보드, 건물 외벽 LED 디스플레이
indoor.jpg   → 실내 회의실, 로비 LED 디스플레이
special.jpg  → 투명 LED, 플렉서블 LED 등
rental.jpg   → 무대, 행사용 이동형 LED
signage.jpg  → 상점 간판, 디지털 사이니지
```

## 임시 이미지

이미지를 준비하기 전까지는 그라데이션 배경과 아이콘이 표시됩니다.
