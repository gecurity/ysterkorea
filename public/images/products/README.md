# 제품 이미지 폴더

이 폴더에 LED 디스플레이 제품 이미지를 저장합니다.

## 사용 방법

1. 제품 이미지를 이 폴더에 저장
2. 관리자 페이지에서 제품 추가/수정 시 파일명만 입력
3. 예: `indoor-led.jpg`, `outdoor-led.jpg`

## 이미지 권장 사양

- **형식**: JPG, PNG, WebP
- **크기**: 가로 1200px 이상 권장
- **비율**: 16:9 또는 4:3
- **용량**: 500KB 이하 권장

## 파일명 규칙

- 소문자 사용
- 띄어쓰기 대신 하이픈(-) 사용
- 예: `indoor-led-p3.jpg`, `outdoor-billboard-p8.jpg`
